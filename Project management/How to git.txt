Push new changes to GitHub:
git add . (Selects all the files in your folder)
git commit -m "Describe you changes" (Selects a branch to push to and adds a description of what you did)
git push (Pushes your changes to GitHub)

Clone repository:
git clone https://link_to_repo (Clones a new repository)


Pull new changes from GitHub:
git fetch
git pull (Pulls changes from an existing repository)
